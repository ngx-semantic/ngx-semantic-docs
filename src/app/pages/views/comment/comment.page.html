Comments

A basic list of comments.

The Semantic UI examples often wrap content in ui segment; that is optional. UI views provide structured layouts but do not wire up site-specific behavior. You will need your own logic to show or hide reply forms—for example, when the user clicks Reply.

Reply form

A comment can contain a form to reply to a comment. Content can be arbitrary. If a comment form is inside a ui comment, it is styled as a nested reply form. If the form appears after all comments, it is styled as a normal reply form.
